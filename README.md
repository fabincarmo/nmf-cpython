Non-Negative Matrix Factorization (NMF) Implementation in C/Python using CBLAS/ATLAS Library
============================================================================================

Cost Function: Kullback-Leibler Divergence.

Requires:
python-dev or python3-dev
libatlas3-base 

Building
--------

```
$ make py2 
```
or
```
$ make py3
```

Example
-------
```
$ python run.py
```
